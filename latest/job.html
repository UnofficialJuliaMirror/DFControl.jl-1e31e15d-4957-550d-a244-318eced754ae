    DFControl.change_atoms! — Method.
    change_atoms!(job::DFJob, atoms::Dict{Symbol,<:Array{<:Point3,1}}, pseudo_set_name=:default, pseudo_specifier=nothing, option=:angstrom)

    Sets the data blocks with atomic positions to the new one. This is done for all calculations in the job that have that data. If default pseudopotentials are defined, a set can be specified, together with a fuzzy that distinguishes between the possible multiple pseudo strings in the pseudo set. These pseudospotentials are then set in all the calculations that need it. All flags which specify the number of atoms inside the calculation also gets set to the correct value.

    DFControl.change_run_command! — Method.
    change_run_command!(job::DFJob, inputnames, run_command)

    Goes through the job calculations and if it contains one of the inputnames it sets the run command of the calculation. Since the run_command field has both the command and the flags, this function only changes the command. To change the run_command's flags use the function set_runflags.

    DFControl.set_flags! — Method.
    set_flags!(job::DFJob, calculations::Vector{String}, flags...; print=true)

    Sets the flags in the calculations to the flags specified. This only happens if the specified flags are valid for the calculations. If necessary the correct control block will be added to the calculation (e.g. for QEInputs).

    The values that are supplied will be checked whether they are valid.
